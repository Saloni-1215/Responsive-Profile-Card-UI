# 👤 Responsive Profile Card UI

## 🚀 About
This project is a responsive profile card UI built using HTML and CSS.  
It demonstrates modern web design techniques including flexbox layout, hover effects, and responsive design.

---

## 🎯 Features
- Clean and modern UI design  
- Responsive layout for different screen sizes  
- Flexbox for alignment and positioning  
- Smooth hover effects  
- Simple and beginner-friendly structure  

---

## 🛠 Tech Stack
- HTML5  
- CSS3  

---

## 📸 Preview
A simple profile card displaying user information such as image, name, role, and description with interactive styling.

---

## 📈 What I Learned
- Structuring web pages using HTML  
- Styling using CSS  
- Using Flexbox for layout  
- Creating hover effects  
- Making responsive designs with media queries
